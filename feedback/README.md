# Feedback Log

Decision history for learning. When Taylor approves or rejects recommendations, log here.

## Format

Each entry:
```json
{
  "date": "2026-02-06",
  "action": "Launch Invoice Tracker on r/Notion",
  "decision": "approved",
  "reason": null,
  "outcome": "pending",
  "lesson": null
}
```

For rejections:
```json
{
  "date": "2026-02-06", 
  "action": "Cold email Fortune 500 CMOs",
  "decision": "rejected",
  "reason": "Too aggressive for current stage",
  "lesson": "Start with warm outreach before cold at scale"
}
```

## Usage

1. Log every Approve/Reject decision
2. For rejections, extract the lesson immediately
3. Weekly: Review patterns, update shared-learnings/

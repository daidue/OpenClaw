## Executive Summary

This research provides **implementation-ready** design specifications for transforming our Notion freelance invoice template into a premium marketplace product ($30-50 tier). Based on analysis of top creators (Easlo, Thomas Frank, Marie Poulin) and marketplace requirements, this document includes:

- Exact hex codes for color palettes
- Specific icon systems with emoji recommendations
- Proven layout patterns with API implementation notes
- Marketplace submission requirements (Notion, NotionEverything, Prototion)
- "Unboxing" experience patterns from $30+ templates

**Key Finding:** Premium templates ($30+) succeed through three elements:
1. **Visual consistency** - Specific color system + icon family (80% use custom emoji sets)
2. **Immediate value** - Users create first invoice within 5 minutes
3. **Guided setup** - Welcome page with video tutorial + sample data

### Cover Image Styles (2025 Trends)

**1. Gradients (40% of premium templates)**
- **Warm Gradients:** Beige ‚Üí Soft Pink, Orange ‚Üí Peach
  - Example hex progression: #F5F1ED ‚Üí #F9F2F5 ‚Üí #FDE8E9

- **Implementation:** Create in Canva, 1500√ó400px, export as PNG

- Abstract shapes (circles, lines, geometric forms)
- Monochromatic or single accent color
- Tools: Figma, Canva, Undraw.co (free)

**3. Photography with Overlay (20%)**
- Unsplash photos with text overlay
- Dark overlay (40% opacity) for text readability
- Contrast ratio: 4.5:1 minimum
- Focus on workspace/productivity themes

**4. Solid Colors with Typography (10%)**
- Single color background
- Large, bold typography
- Minimalist approach
- Example: Navy background (#1A365D) + white text

**Dimensions & Technical:**
- **Minimum:** 1500√ó400px
- **Optimal:** 2400√ó600px (retina)
- **File size:** < 500KB (use tinyjpg.com)

```
- Visual: Largest, bold
- API: page.properties.title

H2 (Major Sections):
- Usage: Main section dividers
- Example: "Active Invoices", "Client Information"
- Spacing: 2-3 empty lines before, 1 after
- Often paired with emoji: "üí∞ Active Invoices"

H3 (Subsections):
- Usage: Subsection headers within H2 sections
- Example: "Payment Terms", "Contact Details"
- Spacing: 2 empty lines before, 1 after
- Can use emoji or text only

Body Text (Regular):
- Usage: All content, descriptions, instructions
- Formatting: Use bold for key terms
- Use italic for secondary notes
- Use inline code for: numbers, IDs, statuses

Quote Blocks:
- Usage: Important callouts, quotes, highlights
- Example: Process descriptions, tips
- Pairs well with emoji prefix

Bulleted Lists:
- Usage: Features, steps, options
- Keep items parallel in structure
- Use sub-bullets sparingly (1 level deep max)

Numbered Lists:
- Usage: Sequential steps, ranked items
- Clear instructions work best
- Can be converted to toggle list for cleaner look
```

**Typography Best Practices:**
- **Bold:** Key terms, important info (sparingly)
- **Italic:** Secondary notes, dates, metadata
- **Color:** Rarely on text (use background color instead)
- **Inline code:** `Status: Paid`, `Invoice #1234` (monospace for data)
- **Underline:** Avoid (indicates links, causes confusion)
- **Strikethrough:** Only for completed items in checklists

## Technical Details

**Architecture:**
- ETL: Daily 8AM ET (Sleeper API + stats + O-line + coaching)
- Pre-Gen: Tuesday 7AM ET (9,900 top player pairs)
- Generation: GPT-5 mini ($0.002/trade)
- Caching: 3-tier (in-memory LRU + PostgreSQL + pre-gen)
- Auth: Required on all endpoints
- Rate Limiting: 10/min generate, 60/min reads

**Database Tables:**
- `player_narrative_context` - Player data from ETL
- `trade_narrative_cache` - Generated narratives (7-day TTL)
- `narrative_generation_log` - Cost/usage tracking

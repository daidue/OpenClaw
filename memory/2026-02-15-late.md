### Critical Patterns Learned
- **🔴 `CI=TRUE` + CRA = build failures** — Every unused import or missing hook dep kills the build. Must lint before pushing.
- **🔴 Cloudflare Pages production branch = `main`** — All code must be on `main` to go live. `master` = preview only.
- **🔴 MailerLite JSONP endpoint returns `success:true` for ANY request** — Even empty ones. Not a reliable indicator of actual subscription.
- **🔴 Hidden iframe = only reliable cross-origin form submission** — `no-cors` fetch, JSONP, and MailerLite's own JS all had issues.
- **🔴 Parallel sub-agent builds create lint debt** — 47 warnings accumulated from 8+ agents building simultaneously.
- **🔴 TypeScript files + CRA + CI=true = build failure** — CRA treats .tsx type errors as warnings → CI=true kills the build. Must convert to .jsx or fix all type issues.
- **🔴 Sub-agent audit output can get truncated** — 200K token limit means complex audits may not return a summary. Check git log for actual work done.
- **🔴 Git submodule ghosts persist after .gitmodules deletion** — Must `git rm --cached` the 160000-mode entries. Cloudflare fails on "No url found for submodule path".

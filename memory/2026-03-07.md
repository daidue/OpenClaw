## Key Decisions & Learnings

**TitleRun's #1 Recurring Bug:** Nested `response.data.data` anti-pattern
- Appeared in 11 PRs since Jan 2026
- Backend sometimes returns `{ data: { data: { ... } } }` (double nesting)
- Frontend expects single envelope: `{ data: { ... } }`
- **Solution:** Enforce single envelope in backend, remove fallback chains in frontend

**useMemo Pattern:** Critical for performance on mobile
- Caused production incident 2026-02-16 (infinite render loop)
- Pattern: Wrap `Map()` creation and regex compilation in `useMemo`
- Mobile Safari re-renders every 16ms → without memoization creates 12K objects/sec

**Test Environment Issues:** Missing `@jest-environment jsdom` directive
- Jest defaults to Node environment (no DOM APIs)
- Causes `ReferenceError: document is not defined`
- **Fix:** Add `/** @jest-environment jsdom */` at top of test files
- Found 358 test failures across repo from same issue (pre-existing, not today's work)

**Agent Orchestration Works:** Rush's Phase 1 fixes validated
- Spawn mechanism functional
- Task file parsing secure
- Lock system prevents race conditions
- All 10 GitHub issues processed successfully

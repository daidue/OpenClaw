### Key Patterns Discovered
- **🔴 Sub-agent integration gaps are the #1 bug source** — Services wire correctly at the service layer but miss endpoint-level data needs (missing columns, undeclared variables)
- **🔴 Hardcoded constants override config changes** — `getAgeFactor()` had `MAX_FACTOR=1.35` that overrode the config anchor update. Always check BOTH config AND code that reads it.
- **🔴 Silent catch blocks hide critical failures** — The TEP catch block logged a warning but returned SF values. In production, these warnings go unnoticed. Consider adding metrics/alerts for TEP fallback frequency.
- **🔴 Heavy overnight work = rate limit risk** — 7+ backend commits + sub-agent spawns + browser automation in one night burned through API credits

- Structure: `PICK_VALUES[year][format][round][position-1]`
- Years: 2024, 2025 (12-team league data)
- Formats: `standard`, `superflex`

**Features:**

   - Converts 8-14 team leagues to 12-team equivalent
   - Formula: `Math.round((position / teamCount) * 12)`
   - Example: 10-team pick 2 → 12-team pick 2

2. **Year delta confidence:**
   - yearDelta = 0 → `HIGH`
   - yearDelta = 1 → `MODERATE`
   - yearDelta ≤ 2 → `LOW`
   - yearDelta > 2 → `ESTIMATED`

3. **Fallback logic:**
   - Uses closest available year if target year missing
   - Returns last value in round if position exceeds 12

4. **Return structure:**
   ```javascript
   {
     value: 5000,
     confidence: 'HIGH',
     source: 'hardcoded',
     yearUsed: 2024,
     originalYear: 2023,
     yearDelta: 1,
     adjustedPosition: 2
   }
   ```

## Technical Anti-Patterns Caught

1. **Math.random() in components** → Seeded PRNG for determinism
2. **Artificial loading states** → CSS transitions instead
3. **Animation resets** → Track current value, animate delta
4. **Missing ARIA linkage** → aria-controls + proper role/state
5. **Text contrast violations** → text-zinc-500 → text-zinc-400
6. **No feature flags** → Environment-based kill switches
7. **Generic error boundaries** → Descriptive + retry actions
